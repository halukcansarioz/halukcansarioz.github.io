Java Tahmin Oyunu

  • Kategori: Java Socket Programing
  • Proje Tarihi: 15 Kasım, 2020
  • Proje URL: https://github.com/halukcansarioz/Socket-Guess-Game-Java

Bu proje okulumda ödev olarak Nesne Yönelimli Programlama dersi için yapılmıştı. Bir istemci/sunucu bağlantısı kullanan Java'da sayı tahmin oyunu. Sunucu oyunu rastgele bir sayı oluşturarak ve kullanıcıdan doğru sayıyı tahmin etmesini isteyerek çalıştırır. Kullanıcı, oyun bitmeden önce sınırlı sayıda tahmin alır. Bir giriş girildiğinde, doğru sayı değilse sunucu "artırma/azaltma" mesajları ile müşteriyi bilgilendirir. Oyun bittiğinde sunucu, kullanıcının oyunu kazanıp kazanmadığını söyler.
